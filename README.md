# lumina-gallery

LUMINA is an immersive 3D art gallery built with pure HTML \& CSS. Explore stunning artworks in a cinematic dark-themed interface featuring smooth animations, floating sculptures, and an elegant editorial layout. A visual experience designed for modern browsers — no frameworks, just code.



\# LUMINA — 3D Art Gallery



An immersive 3D art gallery built with pure HTML \& CSS. Explore stunning artworks in a cinematic dark-themed interface featuring smooth animations, floating sculptures, and an elegant editorial layout.
